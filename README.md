### inno-edu-api

## Build

./gradlew build

## Run

./gradlew bootRun

## Deploy

* Install [Heroku cli](https://devcenter.heroku.com/articles/heroku-cli#download-and-install). 
* Login `heroku loginRequest`.
* Deploy: `git push heroku master`.

You may need to log into the team account.
